### About
AC_program_assignment_2 : Reads a fasta file and find out pairwise alignment scores between each possible pair sequences.

### Usage
python3 AC_program_assignment_2.py

[N.B] Before running the code, place the 'original_sequences_exercise.fasta' file in the coding directory.

### Output
The code returns:
    [i] A list of pairwise alignment scores
    [ii] NxN distance matrix ; here, N is number of possible pairs among sequences
